I Watch Seriez

I Watch Seriez is a file browser to help you keep track of your progress, and much more!

Download this project as a .zip file Download this project as a tar.gz file

Download link:

Latest installer is always available here: https://github.com/csomakk/IWatchSeriez/raw/master/install/IWatchSeriez.exe

I'd prefer the AIR version, which is available here: https://github.com/csomakk/IWatchSeriez/raw/master/install/IWatchSeriez.air and this needs the Adobe AIR runtime available here: https://get.adobe.com/air/

Screenshot:

Features:

  • mark files
  • automatically mark launched files
  • multiple tabs
  • next and last episode of a seriez
  • imdb rating
  • hotkeys

Tips:

Use 'N' keyboard shortcut to mark watched, and start playing next video
